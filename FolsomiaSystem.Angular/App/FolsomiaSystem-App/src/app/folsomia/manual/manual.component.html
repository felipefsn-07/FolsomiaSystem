Pré Contagem

Antes de iniciar o processo de contagem é necessário verificar se a imagem a qual deseja processar a contagem dos Folsomia candida, seguem os padrões de qualidade para realizar um processamento com mais precisão.

A coloração do fundo da imagem de teste (tudo que não remete ao Folsomia candida) deve ser uniforme, evite imagens em que o Folsomia candida não está aparente. Abaixo temos dois exemplos, a primeira imagem representa uma amostra de Folsomia candidacom uma qualidade ruim e que deve ser evitado e no segundo uma imagem na qual os microorganismo estão aparentes.



Para a realização de uma deteção mais eficiente do contador de Folsomia candida, deverá seguir as seguinte premissas:

  • Deverá inserir uma imagem que contenha os Folsomias Candidas a ser identificado.
  • Toda área da imagem desnecessária (tudo que está fora do recipiente que se encontra o solo e os miFolsomias Candidas a ser testado) deverá ser removida através da edição de imagem.
  • O solo testado não deverá ter a mesma coloração do Folsomia candida, ao ponto de o Folsomia candida não ser facilmente identificado.
  • Imagens com baixa resolução tendem a ter uma eficiência de reconhecimento menor, preze em utlizar imagens com resoluções melhores.


Como contonar problemas em que o Folsomia candida não está aparente no solo da imagem?


Uma solução de correção de cor pode ser aplicada na imagem para contornar problemas em que o Folsomia candida não está muito aparente no solo da imagem. A ferramenta Fotos do Windows 10 por exemplo, pode ser utilizada para melhorar a imagem e deixar o Folsomia candida mais aparente.

Com a imagem aberta no aplicativo Fotos do Windows clique em Editar.




Com o Editor de Fotos do Windows aberto, clique em Ajustes, como mostra a figura abaixo:


Assim que o Ajustes for aberto, realize os seguinte ajustes na imagem:

Contador de Folsomia candida

Primeiro deverá ser preenchidos as informações básicas do teste que será realizado.




Assim que os campos obrigatórios forem preenchidos clique no botão . Agora os testes poderão ser iniciados, clique no botão .



O sistema abrirá uma modal onde você podera inserir a imagem e o valor da concentração testada no solo da imagem. Clique no link "Inserir imagem de teste..." para inserir a imagem com Folsomia candida.



Assim que inserida a imagem você poderá editar a imagem e realizar crop (cortar) a imagem, para evidenciar apenas o recipiente com solo.



Finalizado o crop (corte) na imagem, aperte o botão . Agora é só clicar em para iniciar a contagem do Folsomia candida.



Assim que o processo de contagem é finalizado o sistema retornara o total da contagem e uma imagem com os microorganismos Folsomias Candida encontrado circulados.



O processo de contagem pode ser realizados N vezes (a quantidade pode estar limitada pelo adminstrador) e cada vez realizado é adicionado em um pequenos cards como mostra a imagem abaixo:

Editar e remover

editar

Para realizar a edição clique no icone que fica acima do card de concentrações

O sistema abrirá duas opções, o usuário deverá clicar em Editar



O sistema irá abrir a modal de nova concentração e o usuário poderá editar.





Abrir testes salvos

Para realizar a remoção clique no icone que fica acima do card de concentrações

O sistema abrirá duas opções, o usuário deverá clicar em Remover.



Assim que clicar em remover irá aparecer uma mensagem perguntando se deseja remover. Clique em Sim, caso queira remover.

Salvar e abrir

O formato de arquivo salvo e que pode ser aberto posteriormente no Folsomia System é o ".folsys"

Salvar testes

Após finalizar os testes você poderá salvar o trabalho realizado. Para salvar clique no icone de engrenagem

O sistema abrirá um menu com 3 opções, você deverá clicar na opção Salvar para realizar o salvamento de seu teste.





Abrir testes salvos

Para abrir os projetos salvos anteriormente clique clicar no icone de engrenagem

O sistema abrirá um menu com 3 opções, você deverá clicar na opção Abrir para abrir um teste salvo anteriormente.
